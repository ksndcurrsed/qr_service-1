Шаг 1: Подготовка окружения
Прежде всего, убедитесь, что на компьютере установлен Python и все нужные библиотеки.
 * Установите зависимости:
   Откройте терминал и вставьте команду:
   pip install fastapi uvicorn websockets pylibdmtx pillow pywin32 pynput openpyxl

 * Важно для Data Matrix: Библиотека pylibdmtx требует наличия библиотеки libdmtx.dll. Если при запуске возникнет ошибка, скачайте её (Visual C++ Redistributable часто решает эту проблему).

Шаг 2: Узнаем локальный IP компьютера
Чтобы телефон и программа-слушатель знали, куда отправлять данные, нужно найти адрес компьютера в вашей сети.
 * Нажмите Win + R, введите cmd и нажмите Enter.
 * В черном окне введите ipconfig и нажмите Enter.
 * Найдите строку «IPv4-адрес» (например, 192.168.1.50). Запомните его.
Шаг 3: Настройка кодов
В файле server.py (Сервер):
В самом низу убедитесь, что сервер готов принимать внешние подключения:

if name == "main":
    import uvicorn
    # host="0.0.0.0" позволяет заходить на сервер с телефона
    uvicorn.run(app, host="0.0.0.0", port=8000)

В файле main.py (Слушатель):
Найдите строку подключения и замените её на ваш IP:
# Замените 192.168.1.50 на ваш адрес из Шага 2
SERVER_URL = "ws://192.168.1.50:8000/ws-print" 

> Внимание: Для локальной работы используйте ws:// (незащищенный), а для сервера в облаке — wss:// (защищенный).
> 
Шаг 4: Запуск системы
 * Запустите сервер: python server.py
   (Окно должно оставаться открытым. Оно будет показывать логи входящих сканов).
 * Запустите слушатель: python main.py
   (Если всё верно, в консоли появится надпись: "Подключено к серверу!").
Шаг 5: Подключение телефона
 * Убедитесь, что телефон подключен к той же Wi-Fi сети, что и компьютер.
 * В браузере телефона введите адрес: http://192.168.1.50:8000 (ваш IP).
Как включить камеру (Chrome на Android)
Так как мы используем http, а не https, браузер может заблокировать камеру.
 * В Chrome на телефоне перейдите по адресу: chrome://flags/#unsafely-treat-insecure-origin-as-secure
 * В поле вставьте http://192.168.1.50:8000.
 * Нажмите Enabled и затем Relaunch.
 * Теперь при открытии страницы браузер разрешит включить камеру.
